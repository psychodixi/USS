Proyecto dashboard que haga consultas a una API publica sobre Pokémon.

La idea fué buscar algo que nos gustase al grupo, un interes común y agradable para trabajar fueron los Pokémons.
Al buscar API´s públicas, nos encontramos con distintas variedades y de distintas cosas para trabajar, por gusto y limitación nuestra de no querer trabajar con librerías de API´s que ya hicieran mucho por si solas.
tomamos como desafío tomar una API bastante "al fierro", y creamos un dashboard que sea de utilidad y simplicidad para el usuario interesado en los Pokémons.

El proyecto fué realizado con los siguientes estudiantes de la Universidad San Sebastian, todos siendo estudiantes de plan común:
